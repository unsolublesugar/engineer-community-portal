# 01. プロジェクト概要

## サイト名
Engineer Community Portal

## 目的
エンジニアコミュニティの情報総合サイト。
LTイベントの発表資料・動画・登壇者情報を一覧・閲覧できるようにする。

## ページ構成

1. **トップページ** (`/`) - 年度フィルター・ページネーション付きのイベント一覧
2. **イベント詳細ページ** (`/events/NNN`) - 各回の発表一覧、動画埋め込み、スライド埋め込み、前後ナビゲーション
3. **登壇者一覧ページ** (`/speakers`) - 全登壇者のプロフィールと登壇履歴（検索・ソート・ページネーション対応）
4. **企画コーナーページ** (`/challenge`) - 特別企画のアーカイブ
5. **コミュニティ概要ページ** (`/about`) - コミュニティ紹介、運営メンバー、統計情報、参加方法

全ページがクリーンURL（`.html` 拡張子なし）でアクセス可能。

## 技術スタック
- データ管理: YAML（イベントごとに分割）
- ビルド: Python 3.9+ / Jinja2
- ホスティング: GitHub Pages
- CI/CD: GitHub Actions

## デザイン方針
- シンプルかつ見やすいデザイン
- カードベースUI
- レスポンシブ対応（モバイルファースト）
- ライトテーマ基調
- OGP / Twitter Card 対応
